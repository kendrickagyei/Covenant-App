# Chart

This is a React and Vite project set up with hot module replacement and basic ESLint rules. It uses the official Vite plugin for React, powered by Oxc for fast compilation.

The React Compiler is enabled in this template, which can impact both dev and build performance. You can learn more about it in the official React documentation.

If you plan to develop this into a production application, it is recommended to use TypeScript with type-aware lint rules. The Vite TypeScript template provides a good starting point for that integration.
